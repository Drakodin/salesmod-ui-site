mod-panel

A component that houses cards containing all of the recommended product items. It combines a gallery and an endless panel with options for pagination.

Examples

This is a sample panel

Properties

Property Attribute Description Type Default
classes classes Allows the application of custom classes and CSS on-top of existing styles string ""
data data An array of documents or any other value to display on cards in the Panel string undefined
limit limit Sets a limit of cards per page number 0
paginate paginate Applies pagination to the data. Defaults to false boolean false
rounded rounded Generates rounded cards if set to true, otherwise false boolean false
size size Selector for mounted card sizes "large" \| "medium" "medium"
theme theme Theme used to influence the cards within the panel "abyss" \| "autumn" \| "breeze" \| "minimalist" "minimalist"
